Creates a Context from supported input values

The supported values are essentially used to enable supported controllers/controller configs via hid commands, and for opening Player types

Arguments
  • supported_style_tags: Supported NpadStyleTag flags
  • supported_npad_ids: Supported NpadIdType values
source

pub fn get_player(&self, npad_id: NpadIdType) -> Player

Opens a Player type for the specified NpadIdType

This simplifies creating a Player type, since this context contains the supported NpadStyleTag values and the mapped shared-memory address

Arguments

npad_id: The NpadIdType to use
